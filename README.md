Данная версия консоли является сборкой из всех модулей `Пихты` (`pht`/`phtx/pls`).<br>
Версия адаптирована для демонстрации примеров из папки `examples`.<br>
*Иногда в примере **важен не вывод**, **но код** самого примера.*
